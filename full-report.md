# AIエージェント比較実験 全データレポート

6つのAIコーディングエージェント（Claude Code・Codex CLI・Antigravity CLI・Codex IDE・Antigravity IDE・Copilot Agent）の、実験A・B・D・Eにおける定量データ・定性評価・コメント全文をまとめたものです。

「総合的に優れたAIエージェントはどれか」という問いには単一の正解がありません。実装力を重視するか、誠実さを重視するかで、結論は変わります。本レポートのデータだけで、どれかのエージェントに優劣をつけられるものではなく、ご自身の用途に合わせて読み解いていただくための資料です。

- 定性評価の「ギャップ」は `自己評価 - 人間評価` で、正の値は過大評価、負の値は過小評価を示します。

- コメントは `evaluation.json` の `notes` フィールド、および各エージェントが記録した実験ノートの内容をそのまま転載しています。
