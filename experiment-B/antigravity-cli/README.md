# TaskFlow - モダンタスク管理Webアプリ

TaskFlowは、FastAPI + SQLiteの堅牢なバックエンドと、Vue 3 (CDN) + Vanilla CSSによる洗練されたデザインのフロントエンドを組み合わせた、シングルページのタスク管理（カンバンボード）アプリケーションです。

---

## 主な機能

1. **直感的なカンバンボードUI**:
   - タスクを「未着手」「進行中」「完了」の3つのステータスで管理。
   - HTML5 Drag and Drop API を使用した、カードのドラッグ＆ドロップによる直感的なステータス移動（楽観的UI更新によるスムーズな操作感）。
2. **タスクのCRUD操作**:
   - モーダルによるタスクの作成・編集。
   - タイトル（必須）、詳細、期限日、優先度（低・中・高）の設定。
   - タスクの削除機能（確認ダイアログ付き）。
3. **検索とフィルタリング**:
   - タイトル・詳細テキストのキーワード検索（デバウンス処理により、入力中の過剰なAPIリクエストを防止）。
   - 優先度（高・中・低）による動的フィルタリング。
4. **期限切れタスクの視覚的強調**:
   - 期限日を過ぎた未完了タスクには「期限切れ」バッジを表示し、カード左側を警告色（赤）でハイライト。
5. **モダンなダークテーマデザイン**:
   - 半透明の背景（Glassmorphism）や滑らかなホバーエフェクト。
   - 操作成功・失敗時のスムーズなフェードイン付きトースト通知。

---

## システム構成

- **バックエンド**: Python 3.13 + FastAPI + SQLAlchemy + SQLite
- **フロントエンド**: Vue 3 (CDN) + Vanilla CSS (HTML5 Drag and Drop API)
- **テスト**: pytest + httpx (FastAPI TestClient)

---

## セットアップと起動手順

### 1. 依存ライブラリのインストール

Python 3.13 環境で以下のコマンドを実行し、必要なパッケージをインストールします。

```bash
python -m pip install -r requirements.txt
```

### 2. アプリケーションの起動

Uvicorn を使用して FastAPI サーバーを起動します。

```bash
python -m uvicorn main:app --reload --port 8000
```

起動後、ブラウザで以下のURLを開いてください。
- アプリケーション画面: [http://localhost:8000](http://localhost:8000)
- APIインタラクティブドキュメント (Swagger UI): [http://localhost:8000/docs](http://localhost:8000/docs)

※ データベース（`tasks.db`）はアプリケーションの初回起動時に自動的に生成されます。

---

## APIテストの実行手順

FastAPI の `TestClient` を用いた API のユニットテストを実行するには、以下のコマンドを実行します。

```bash
pytest
```

テスト実行時はインメモリデータベース (`sqlite:///:memory:`) が使用されるため、本番の `tasks.db` のデータが影響を受けることはありません。
